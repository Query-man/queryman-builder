=== Installation
==== Gradle
Add the following block into `build.gradle`:
```
repositories {
    mavenCentral()
}

dependencies {
    compile group: 'org.queryman', name: 'queryman', version: '{version}'
}
```

==== Maven
If you use maven, add this block into `pom.xml`:
```
<dependency>
  <groupId>org.queryman</groupId>
  <artifactId>queryman</artifactId>
  <version>{version}</version>
</dependency>
```

=== Configuration
The configuration xml file named `queryman-configuration.xml` can to be place to `resources` directory.
More information about XML Schema you can find in {domain}/xsd[official site]. The details of configuration
